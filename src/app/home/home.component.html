This website uses Angular in the front-end to request all the projects from a RESTful web service API with an Http Get method and renders the JSON response in a list and a map.
In the back-end an AWS API Gateway executes a serverless Lambda function in Node.js that reads data from DynamoDB. The domain is registered in Route 53 and hosted in S3 bucket. The HTTP/SSL certificate is distributed using CloudFront.
