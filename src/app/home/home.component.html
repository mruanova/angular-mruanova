This website in the front-end uses Angular
to request all the projects from a RESTful
web service with an Http Get method and
renders the JSON response in a list and a map.

In the back-end an AWS API Gateway endpoint
executes a serverless Lambda function in Node.js
that reads data from DynamoDB.

The domain is registered in Route 53
and the website is hosted in an S3 bucket.
CloudFront distributes the HTTPS/SSL certificate.
